- (Object) index

Url:

  • [GET] /v0/contacts
  • [GET] /v0/accounts/:account_name/contacts

Returns list of contacts in JSON

Optional Arguments:

  • (Array) ids —

    return contacts with id in this array

  • (String) account_name —

    will scope contacts to this account

  • (String) list_name —

    scope to this list. Will be ignored if no :account_name is given.

  • (Integer) page —

    will return this page (default: 1)

  • (Integer) per_page —

    will paginate contacts with this amount per page (default: 10)

  • (String) full_text —

    will make a full_text search with this string.

  • (Hash) where —

    Mongoid where selector with additional keys -> :email, :telephone, :address, :local_status

Example Response:

{ collection: [ {_id: 1234,name: ...} ], total: 1}

Response Fields:

  • (Array <Contact>) collection —

    corresponding to chosen :page

  • (Integer) total —

    total amount of contacts in query. (includes all pages.)

- (Object) show

Url:

  • [GET] /v0/contacts/:id
  • [GET] /v0/accounts/:account_name/contacts/:id

Returns JSON for a contact if account is provided following attributes will be inclueded:

* owned by account
* public attributes
* masked attributes

Required Arguments:

  • (String) id —

    contact_id

Optional Arguments:

  • (String) account_name —

    scope search to this account. Fields will be added to response when this is sent.

Example Response:

if account_name is provided
  {id: '124365w45215', first_name: 'Dwayne', last_name: 'Macgowan'}
else
  {id: '123', first_name: 'Dwa', last_name: 'Mac', linked: true}

Response Fields:

  • (TrueClass) linked —

    is this contact linked to :account_name?

  • (String) first_name
  • (String) last_name
